### nest.js
```
yarn global add @nestjs/cli
```

npx nest new next-introduction

### prisma

```
yarn add -D prisma
yarn add @prisma/client
```

```
npx prisma init
```

```
npx prisma migrate dev
```
→ schema.prismaからmigrationを実行

```
npx prisma studio
```

```
npx prisma generate
```
